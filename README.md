🧵 DesiLooms

A handloom e-commerce web application dedicated to promoting and selling traditional Indian woven products. DesiLooms connects artisans with customers, preserving the rich heritage of Indian handloom culture.
🌟 Features

🏠 Home Page — Browse featured handloom products
🛍️ Products Page — View full catalogue of woven goods
🛒 Cart — Add items and manage your order
📦 Order & Order Confirmation — Seamless ordering experience
🔐 Login & Register — User authentication system
👑 Admin Panel — Manage orders and add new products

🛠️ Tech Stack

Frontend: HTML, CSS
Backend: Python (Flask)
Database: SQLite

▶️ How to Run

Install dependencies:

   pip install flask

Run the app:

   python app.py

Open your browser and go to:

   http://localhost:5000
